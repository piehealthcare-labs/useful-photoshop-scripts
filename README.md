# 유용한 포토샵 스크립트

업무 효율을 높여주는 포토샵 스크립트 4종 모음입니다.

## 포함된 스크립트

1. **A 아트보드별 저장.jsx**
   - 현재 문서의 각 아트보드를 개별 파일로 빠르게 저장합니다.
2. **B 열려있는 파일 모두 jpg 내보내기.jsx**
   - 포토샵에 열려있는 모든 문서를 일괄적으로 JPG 파일로 내보냅니다.
3. **C 아트보드를 캔버스로.jsx**
   - 아트보드를 일반 캔버스 형태로 변환하여 레이어 구조를 유지합니다.
4. **D 눈 감은 레이어 삭제.jsx**
   - 가시성이 꺼진(눈을 감은) 불필요한 레이어들을 현재 파일에서 한 번에 삭제합니다.

## 사용 방법

1. 포토샵을 실행합니다.
2. 상단 메뉴바에서 **[파일 (File)]** > **[스크립트 (Scripts)]** > **[찾아보기... (Browse...)]** 를 클릭합니다.
3. 실행할 `.jsx` 파일을 선택하면 스크립트가 실행됩니다.
4. 자주 사용하는 스크립트는 포토샵이 설치된 폴더 내의 `Presets/Scripts` 경로에 넣으면 메뉴에 바로 표시되어 더 편리하게 사용할 수 있습니다.
